Project snapshot

What's inside

The four assignment pages sit inside this single site as iframes, so you can open PSI data, SVG logos, D3 sketches, and the adversarial heatmap without leaving the page.

How I merged them

I kept every original index.html untouched in its folder and embedded each one into the main layout. This kept their CSS/JS isolated while giving the site one navigation bar and shared styling.

Difficulties

Assignments 3 and 4 refused to show up at first because they were not being included in the deployed bundle.

How it was solved

I updated the deployment copy step to include every assignment folder and re-tested the HTML linting so the GitHub Pages workflow could deploy the full site.
